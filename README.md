# Whisper GPU Builder

یک مخزن آماده برای اجرای Whisper روی GPU با گیت‌هاب اکشن.

## چطور استفاده کنم؟

1. روی دکمه **"Use this template"** کلیک کن
2. برو به تب **Actions**
3. workflow **"Build Whisper with GPU and Process Audio"** رو انتخاب کن
4. روی **"Run workflow"** کلیک کن
5. مدل مورد نظرتو انتخاب کن (`base` برای شروع مناسب است)
6. (اختیاری) لینک مستقیم فایل صوتی را وارد کن
7. صبر کن تا اکشن تموم بشه
8. فایل `whisper-results.zip` رو از بخش **Artifacts** دانلود کن

## فایل‌های خروجی

- `result.json` - شامل تایم‌های دقیق کلمه‌ها
- `result.txt` - متن خام

بدون نیاز به نصب هیچ چیزی روی سیستم خودت!
